# 🌿 Plant Disease Detection

A Machine Learning based Plant Disease Detection system that identifies diseases in plant leaves using image dataset patterns and classification models. This project helps farmers and agricultural professionals detect plant diseases early, reduce crop damage, and improve productivity.

---

## 🚀 Features

- Predict plant diseases from leaf images
- Machine Learning based classification
- Fast and accurate disease detection
- Simple and user-friendly interface
- Helps in early crop disease management
- Supports multiple disease categories

---

- **NumPy**

## Installation
1. Clone the repository:
   
git clone https://github.com/itsharshnagar/Plant-Disease-Detection.git
cd Plant-Disease-Detection
   

2. Create and activate a Python virtual environment:
   
   python -m venv env
   env\Scripts\activate
   

3. Install required packages:
   
   pip install -r requirements.txt
   


4. Run the application:
   
   python app.py
